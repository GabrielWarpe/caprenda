O que é a linguagem C?

C é uma linguagem de programação de propósito geral criada em 1972 por Dennis Ritchie nos Laboratórios Bell. Nasceu para reescrever o UNIX de forma portável e se tornou a base de praticamente toda a computação moderna: Linux, Windows, Python e Java foram escritos (total ou parcialmente) em C.

Por que aprender C?
  • Você entende memória, ponteiros e desempenho de verdade
  • Facilita aprender C++, Java, Python e outras linguagens
  • Muito usado em sistemas embarcados, jogos e sistemas operacionais
  • Exigido em muitos processos seletivos e disciplinas universitárias

Compilada vs. Interpretada

C é compilada: você escreve o código-fonte, o compilador transforma em código de máquina (binário), e a CPU executa esse binário diretamente — por isso C é muito rápido.

Código fonte (.c) → GCC → Binário → CPU → Resultado

Estrutura de um programa C

Todo programa C tem obrigatoriamente a função main() — o ponto de entrada onde a execução começa.

main.c
#include <stdio.h>   // 1. Importa biblioteca de entrada/saída

int main() {             // 2. Função principal (obrigatória)
    printf("Olá!\n");   // 3. Imprime na tela (\n = nova linha)
    return 0;           // 4. 0 = programa terminou com sucesso
}

Compilando com GCC

terminal
# Compilar
gcc main.c -o meu_programa

# Executar
./meu_programa

O que são variáveis?

Uma variável é um espaço nomeado na memória RAM onde guardamos um valor durante a execução do programa. Em C, toda variável precisa ser declarada com um tipo antes de ser usada.

Tipos primitivos em C

TipoTamanhoFaixa aproximadaUso

float4 bytes6–7 dígitos decimaisDecimais simples
double8 bytes15–16 dígitos decimaisDecimais precisos
char1 byte0 a 255 (ASCII)Um caractere
long8 bytesInteiros grandesContadores grandes

Declarando e inicializando variáveis

declaracao.c
int idade = 20;
float altura = 1.75;
double pi = 3.14159265358979;

if / else if / else

Permitem executar blocos de código apenas quando uma condição é verdadeira. Em C, 0 é falso; qualquer outro valor é verdadeiro.

condicional.c
int nota = 7;
if (nota >= 9) {
    printf("Excelente!\n");
} else if (nota >= 7) {
    printf("Aprovado!\n");   // cai aqui
} else {
    printf("Reprovado\n");
}

OperadorSignificadoExemplo

switch — múltiplas opções

switch.c
int dia = 3;
switch (dia) {
    case 1: printf("Segunda\n"); break;
    case 3: printf("Quarta\n");  break;
    default: printf("Outro dia\n");
}

Laço for

for.c
// Imprime 1 até 5
for (int i = 1; i <= 5; i++) {
    printf("%d ", i);   // 1 2 3 4 5
}

while e do-while

while.c
// while: verifica ANTES de executar
int i = 1;
while (i <= 5) {
    printf("%d\n", i++);
}

// do-while: executa PELO MENOS UMA VEZ
int senha;
do {
    printf("Digite 1234: ");
    scanf("%d", &senha);
} while (senha != 1234);

O que são funções?

Funções são blocos de código nomeados e reutilizáveis que realizam uma tarefa específica. São o principal mecanismo para organizar e reutilizar código em C.

Estrutura de uma função: 
tipo_retorno nome_funcao(tipo param1, tipo param2) {
    // corpo da função
    return valor;
}

Protótipos, definição e chamada

funcoes.c
#include <stdio.h>

// Protótipos
int soma(int a, int b);
void imprimirLinha();

int main() {
    imprimirLinha();
    printf("Soma: %d\n", soma(10, 5));
    return 0;
}

int soma(int a, int b) { return a + b; }
void imprimirLinha() { printf("----------\n"); }

Passagem por valor vs. por referência

referencia.c
// Por referência — altera o original
void dobra(int *n) {
    *n = *n * 2;
}
int main() {
    int x = 5;
    dobra(&x);
    printf("%d\n", x);  // 10
    return 0;
}

Recursão

recursao.c
int fatorial(int n) {
    if (n <= 1) return 1;   // condição de parada
    return n * fatorial(n - 1);
}
// fatorial(5) = 120
